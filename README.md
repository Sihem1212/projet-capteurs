# Projet Capteurs

## Description
Ce projet simule la collecte et l'affichage de données provenant de capteurs (température, luminosité, distance) sur un site web.

## Technologies utilisées
- HTML et CSS pour l’interface web
- PHP pour la génération dynamique des données
- Simulation de capteurs avec valeurs aléatoires

## Fonctionnalités
- Page principale affichant les mesures simulées
- Interface simple et lisible
- Projet facilement extensible pour intégrer de vrais capteurs

## Objectif
Montrer mes compétences en développement web et en intégration de données de capteurs.
